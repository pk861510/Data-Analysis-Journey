## Overview
This project is part of my Data Analysis journey, focusing on extracting and analyzing data from the YouTube Data API. The goal is to uncover trends in viral videos, specifically looking at engagement metrics like view counts, like/dislike ratios, and comment sentiment.

## Key Features
* **Secure API Integration:** Connects to the YouTube Data API v3 using Python.
* **Data Extraction:** Retrieves channel statistics and trending video data.
* **Data Visualization:** (In Progress) Scatter plots analyzing the correlation between likes, dislikes, and views.
* **Security:** Uses environment variables to protect sensitive API keys.

## Tech Stack
* **Python** (Data extraction & analysis)
* **Google YouTube Data API v3**
* **Pandas** (Data manipulation)
* **Matplotlib / Seaborn** (Visualization)
* **Python-dotenv** (Environment security)
